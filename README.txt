GrMPy: A Python implementation of graphical models.
by Almero Gouws (14366037@sun.ac.za), Helge Reikeras <helge@sun.ac.za>.

The examples folder contains tutorial .py files illustrating the use of all the possible functions of GrMPy. For example ./Examples/Discrete/MRF/Inference/Exact contains examples of performing exact inference on Markov random fields.
